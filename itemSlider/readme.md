
##Item Slider

Item slider direvtive for hybrid application.
```html
	<item-slider data="sliderData"></item-slider>
```

1. Use item slider as element.

2. Pass your data into data attribute. It should be and array with json format.

examples:-
```javascript
$scope.sliderData =[
					{
						id    : 1,
						name  : 'test',
						image : 'demo.jpg'
					},
					{
						id    : 2,
						name  : 'test',
						image : 'demo.jpg'
					},
					{
						id    : 3,
						name  : 'test',
						image : 'demo.jpg'
					},
					{
						id    : 4,
						name  : 'test',
						image : 'demo.jpg'
					}
				] 
```
